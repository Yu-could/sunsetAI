## 后期制作流程

### 1. 素材整理
- 收集所有录屏素材
- 整理现场拍摄素材
- 准备Logo动画和文字动画

### 2. 剪辑顺序
```
痛点引入 → 产品亮相 → 核心功能演示 → 方言识别 → 子女端展示 → 结尾
```

### 3. 音效添加
- 背景音乐：温暖、舒缓风格
- 点击音效：按钮点击、界面切换
- 提醒音效：铃声、警报
- 成功音效：识别成功、数据录入

### 4. 字幕添加
- 关键功能点添加字幕
- 字幕位置：画面下方
- 字幕字体：黑体，白色描边

### 5. 转场效果
- 镜头间使用淡入淡出
- 功能切换使用滑动转场
- Logo出现使用渐现效果

## 3. Toast 消息系统（替换 alert）

**目标**：用优雅的顶部弹出消息替换生硬的浏览器 `alert()` 弹窗。

**新建文件**：
- `src/stores/toastStore.js` — Pinia 状态管理
- `src/components/Toast.vue` — 全局 UI 组件

**Toast 类型与样式**：
```
success → 绿色边框 + ✅  2秒自动消失
error   → 红色边框 + ❌  3秒
warning → 琥珀边框 + ⚠️  2.5秒
info    → 蓝色边框 + ℹ️  2.5秒
```

**集成到 App.vue**：`<Toast />` 放在模板顶部，`<Teleport to="body">` 挂载到 body。

**已替换 alert 的页面**：Home.vue（签到）、SOS.vue（求助/呼叫）、Health.vue（数据验证/保存）。

**用法示例**：
```js
import { useToastStore } from '../stores/toastStore'
const toast = useToastStore()
toast.success('签到成功！', 2000)
```

## 5. 智能用户头像

**目标**：用户填写实名信息后，头像自动从通用图标变为姓名首字或上传的照片。

**新建文件**：`src/components/UserAvatar.vue`

**显示优先级**：
```
1. 自定义上传图片 (base64) → <img> 圆形裁剪
2. 姓名最后一个字      → 渐变背景 + 白色文字 (如 "张小明"→"明")
3. 通用人形图标        → UserIcon
```

**角色配色**：
- parent → 暖橙渐变 `from-parent to-parent-hover`
- child → 冷蓝渐变 `from-child to-child-hover`

**已集成位置**：
- `Home.vue` 移动端头部横幅右侧
- `desktop/Home.vue` 桌面端头部横幅右侧
- `desktop/Layout.vue` 桌面端侧边栏底部用户区

**头像点击跳转**：3 处头像均包裹 `<router-link to="/realname">`，点击进入个人主页。

## 11. Bug 修复

### Bug 1：子女端导航显示错误
- **问题**：子女端页面底部显示父母端菜单
- **原因**：`currentNavItems` 判断用了 `window.location.pathname`，Hash 模式路由下始终返回 `/sunsetAI/`，无法识别 `/child` 路由
- **修复**：改为 `route.path.startsWith('/child')`

### Bug 2：部署后页面空白
- **问题**：GitHub Pages 只显示标题，JS 资源 404
- **原因**：Windows Git Bash 把 `base: '/sunsetAI/'` 解析为 `/git/Git/sunsetAI/`，提交了错误路径的 dist 文件
- **修复**：`dist/` 不再提交到仓库，由 GitHub Actions CI (Ubuntu) 构建 → 路径正确

### Bug 3：页面标题乱码
- **问题**：浏览器标签显示"澶曢槼鏅鸿"
- **原因**：`index.html` 中文字符编码损坏
- **修复**：`<title>夕阳智语 - SunsetAI</title>`

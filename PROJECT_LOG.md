## 历史功能

### 核心功能模块
- 健康监测（Health.vue）
- 用药提醒（Medicine.vue）
- 回忆录（Memories.vue）
- 饮食管理（Diet.vue）
- 平安签到（Safety.vue）
- 亲情连接（Family.vue）
- 日程提醒（Schedule.vue）
- 紧急求助（SOS.vue）

### 用户角色
- 父母移动端（/parent）
- 子女移动端（/child）
- 父母桌面端（/desktop）
- 子女桌面端（/desktop/child）
- 管理员后台（/admin）

### 技术栈
- Vue 3 + Vite
- Tailwind CSS
- Vue Router（Hash 模式）
- Pinia（状态管理）
- GitHub Pages（静态部署）
